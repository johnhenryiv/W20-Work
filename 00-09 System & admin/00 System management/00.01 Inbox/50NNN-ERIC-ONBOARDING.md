# 50NNN Launch — Eric's Quick Start Guide

Welcome! You're collaborating on an exciting project launching this summer. Here's everything you need to access and get started.

---

## Your Access Points

### 1. Bitwarden (Password Manager)
**What:** Secure shared passwords and credentials  
**How to access:**
- Go to bitwarden.com
- Sign up with your email (you'll get an invite from the project lead)
- Set a strong master password
- Enable two-factor authentication (2FA)
- Accept the organization invite: `50NNN-Launch`

**What you'll find:**
- Synology login credentials
- Podcast hosting credentials
- Partnership contact info
- Any shared passwords needed for the project

---

### 2. Synology (Shared File Storage)
**What:** Central hub for all project files (audio, video, graphics, production notes)  
**How to access:**
1. You'll receive Synology login credentials via Bitwarden
2. Go to the Synology URL provided
3. Log in with username: `eric` and the password from Bitwarden
4. **On first login:** Change your password to something only you know
5. Download and install **Synology Drive app** on your computer
6. Sync the `50-single-project` folder to your local drive for easy access

**Folder structure you'll see:**
```
50-single-project/
├── podcast-episodes
├── scripted-segments
├── quick-stories
├── graphics-assets
├── audio-files
├── production-notes
└── contacts-partnerships
```

---

### 3. Google Drive (Planning & Documentation)
**What:** All project planning, outlines, guest info, and timelines  
**How to access:**
- You'll get a shared link to `50NNN Launch` folder
- Open it and click **"Add to My Drive"**
- You'll have Editor access — create, edit, and organize as needed

**Key docs you'll use:**
- **Master Production Doc** — Project overview & status
- **Guest Host Database** — Guest info & availability
- **Episode Ideas** — Podcast episode concepts
- **Segment Ideas Log** — Scripted segment & story concepts
- **Production Calendar** — Timeline & task tracker

---

### 4. Google Keep (Shared Notes)
**What:** Quick brainstorming, daily notes, ideas  
**How to access:**
- Shared notebook: `50NNN-Launch-Notes`
- Use for quick captures before they go into formal docs
- Jot down ideas, reminders, funny moments

---

## Your First Week Tasks

1. **Set up Bitwarden**
   - Create account, enable 2FA
   - Accept organization invite
   - Save Synology credentials from shared vault

2. **Set up Synology**
   - Log in and change your password
   - Install Synology Drive app
   - Sync the `50-single-project` folder locally

3. **Set up Google Drive**
   - Add `50NNN Launch` folder to your Drive
   - Open and skim Master Production Doc & Guest Host Database
   - Familiarize yourself with folder structure

4. **First Check-In Call**
   - Schedule a 30-minute kickoff with the project lead
   - Confirm you can access all systems
   - Discuss your role & initial tasks
